# A simple cli tool

### Commands available:

- tool --build
- tool --start
- tool --help
- tool --versiontool --build

### Example:

```bash
tool --build
```

### Output:

```
Building the app...
```

### Requirements:

- Node.js 14 or higher

### Installation:

```
bash
#/tool + /cli-tool-project

npm i
```
